easyHTTP Library

This app represents a custom AJAX library similar in function to Express, is built with vanilla JS ES5, and refactored into JS ES6. All original and refactored versions of the JS files are stored explicitly for the sake of easy comparison (contra typical Git procedure). The app provides simplified CRUD commands in the associated ES5 easyhttp.js, which are then illustrated in the app.js file. The refactored ES6 versions are enumerated; e.g., easyhttp3.js and app3.js files. JSONPlaceholder is used as our RESTful API for mimicking a server and data manipulation. The results are displayed in the console.
