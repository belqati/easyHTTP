easyHTTP Library

This app represents a custom AJAX library similar in function to Express, and is built with vanilla JS ES5, refactored into JS ES6. It provides simplified CRUD commands in the associated easyhttp.js, which are then illustrated in the app.js file. The refactored version is found in the easyhttpES6.js and appES6.js files. JSONPlaceholder is used as our RESTful API for mimicking a server and data manipulation. The results are displayed in the console.
